🗳️ Smart Election Voting System Using Face Recognition

📌 Project Overview

The Smart Election Voting System Using Face Recognition is a secure and intelligent web-based voting application designed to modernize the election process. Instead of traditional voter verification methods, the system authenticates voters through facial recognition, ensuring that only authorized individuals can cast their vote.

The project aims to improve election security, eliminate duplicate voting, reduce manual verification, and provide an efficient digital voting experience.

---

🎯 Objectives

- Develop a secure online voting system.
- Authenticate voters using face recognition technology.
- Prevent duplicate and unauthorized voting.
- Simplify election management through an admin dashboard.
- Store voter information securely using MongoDB.

📂 Project Workflow

1. Admin logs into the system.
2. New voters are registered with their facial data.
3. The admin starts the election.
4. The voter is authenticated using face recognition.
5. Eligible voters cast their vote securely.
6. Duplicate voting is automatically prevented.
7. The admin can stop the election and manage the voting process through the dashboard.
